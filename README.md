# 📌 Countries101

## ⚙️ How to run

### 1. Install the dependencies

`npm i`

### OR

`yarn add`

## 2. Environment Variables

### You need to config some variables in order to run the app:

`NEXT_PUBLIC_API_URL`

## 3. Run the project

### Run the project using:

`npm run dev`

### OR

`yarn dev`
